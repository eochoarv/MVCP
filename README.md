# MVCP
This repository helps to reproduce the experiments to evaluate Multi-view Conformal Prediction on IR and Raman spectroscopy data

## Getting Started

This package is self-contained and implemented in python.

### Prerequisites

* python
* numpy
* scipy
* scikit-learn
* pandas

### Installing

```bash
git clone git@github.com:eochoarv/MVCP.git
```

## Usage

Please refer to [usage_example.ipynb](usage_example.ipynb) for basic usage. 
Numerical experiments can be found in [numerical_experiments.ipynb](numerical_experiments.ipynb).
